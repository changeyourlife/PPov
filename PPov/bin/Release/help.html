Программное обеспечение "Обменный пункт" создал Негода Андрей

  • Программное обепечение служит для обновления курса валют из Национального банка Украины по адресу http://www.bank.gov.ua;
  • Для использования программного обеспечения нужен доступ в интернет;
  1. Нажмите на кнопку "Обновить" для того чтобы сформировать таблицу со списком валют (список формируется из XML файла, который находится по адресу http://pfsoft.com.ua/service/currency/);
    • Если программа выдаёт ошибку, то это значит, что сервер не пропустил запрос и надо попробовать ещё раз запросить;
  2. Выпадающими списками можно выбрать валюты, которыми хотите оперировать;
    • Перечень сокращений используемый в программе:
      1. AUD - австралийский доллар
      2. AZN - азербайджанский манат
      3. GBP - английский фунт стерлинга
      4. BYR - белорусский рубль
      5. DKK - датская крона
      6. EUR - евро
      7. ISK - исландская крона
      8. KZT - казахстанская тенге
      9. CAD - канадский доллар
      10. LTL - литовский лит
      11. MDL - молдавский лей
      12. NOK - норвежская крона
      13. PLN - польский злат
      14. RUB - российский рубль
      15. SGD - сингапурский доллар
      16. XDR - СПЗ
      17. TRY - турецкая лира
      18. TMT - туркменский манат
      19. HUF - венгерский форинт
      20. UZS - узбекский сум
      21. CZK - чешская крона
      22. SEK - шведская крона
      23. CNY - китайский юань
      24. JPY - японский иен
  3. Вписав в текстовое поле число и нажав на кнопку "Рассчитать", программа покажет вам рассчёт по выбранным вами параметрам;
  4. Нажав на кнопку "Сохранить информацию" вы сохраните список валют предоставленный в таблице, в текстовом варианте. Текстовый файл сохранится туда где лежит исполняемый файл и будет называться числом, которым была обновлена база Нацбанка Украины;
